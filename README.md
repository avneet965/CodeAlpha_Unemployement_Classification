### Iris Flower Classification

CodeAlpha Data Science Internship Project

### Objective

Build a machine learning model to classify Iris flowers into:

* Iris-setosa

* Iris-versicolor

* Iris-virginica

### Technologies Used

* Python

* Pandas

* NumPy

* Matplotlib

* Seaborn

* Scikit-Learn

### Models Compared

* Logistic Regression

* Decision Tree

* Random Forest
